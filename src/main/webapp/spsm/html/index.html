秸秆“废物利用”后的价值裂变


如果压制成生物燃料，产生的热量 与一吨3500大卡标煤相当；如果加工成高能饲草， 则相当于50头奶牛一天的口粮；送去生物质电厂， 则可产生500度电…秸秆禁烧与综合利用是硬币的两面， 在禁烧的同时，如何给秸秆找出路，让秸秆发挥应有的价值， 则更具有长远的现实意义。

小麦品种更新换代工程加快了农业科技成果转化速度


根据北京市小麦生产需求，在引种工作上坚持“丰产品种与优质品种兼顾， 高水肥品种与节水品种兼顾”的原则，积极引进最新育成的各种类型优良小麦新品种。这些品种主要来源于中国农科院、中国农业大学等在京农业科研院所和高校，以及在京种子企业、外省育种单位等，代表了北京乃至北部冬麦区最先进的小麦育种水平。

    • 智能LED植物工厂是我国近五年来的重大科技成果,使我国成为世界上少数掌握该技术的国家之一。 这类技术通过光环境模拟与LED智能控制、立体多层栽培、营养液循环和品质控制,让光照、温度、水分、营养、二氧化碳等植物所需的条件都由人工提供,技术人员可通过手机和平板电脑随时随地了解植物生长情况。 广东现代农业应用示范基地LED植物工厂产出的蔬菜成本就要1块多钱一棵,比市场上的有机无公害蔬菜还要贵出不少。 此外,植物工厂设施投入不菲,每平方米投入约为数千至一万元。


    种业是关系国计民生的战略性、基础性产业，是现代农业发展的生命线。2010年10月，北京市农委、市农业局等部门联合发布了《北京种业发展规划（2010-2015）》，提出要不断提高首都种业的影响力、竞争力、控制力和吸引力，打造“种业之都”。 在《规划》的指导下，北京市实施了“2468种业行动计划”，以及小麦、玉米、蔬菜、马铃薯4个种子产业化工程，进一步巩固了全国种业科技创新中心、企业聚集中心、交易交流中心和发展服务平台——“三中心一平台”的地位。
